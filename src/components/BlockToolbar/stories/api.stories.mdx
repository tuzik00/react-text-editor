## Example

**buttons** - массив кнопок, состоит из type (тип блока), (label выводится как подсказка
и если не передан icon используется как текст кнопка) и icon (иконка в кнопке)

```typescript jsx
import React from 'react';
import { BlockToolbar } from '@citilink/content-editor';
import { IconPhoto } from '@/components/Icons';

const Component = () => (
    <BlockToolbar
        buttons={[
            {
                type: 'photo',
                label: <IconPhoto />,
                title: 'Добавить фото',
            },
            {
                type: 'gallery',
                label: 'Гелерея',
                title: 'Добавить галерею',
            },
        ]}
    />
);

export default Component;
```


**renderButton** - каждая кнопка рендерится через этот пропс, если нужно будет добавить какие-то стилизации,
это можно сделать в renderButton

```typescript jsx
import React from 'react';
import { BlockToolbar } from '@citilink/content-editor';

const Component = () => (
    <BlockToolbar
        renderButton={(element, button) => {
            if (button.type === 'gallery') {
                return (
                    <div>
                        {element}
                    </div>
                );
            }

            return element;
        }}
    />
);

export default Component;
```
