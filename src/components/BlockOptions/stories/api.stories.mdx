## Example

**itemsList** - массив кнопок

```typescript jsx
import React from 'react';
import { IconDelete } from '@/components/icons';

import { BlockOptions } from 'react-txt-editor';

const Component = () => (
    <BlockOptions
        itemsList={[
            {
                title: 'Delete',
                icon: <IconDelete />,
                action: () => {},
            }
        ]}
    />
);

export default Component;
```


**isShow** - Флаг для скрытия меню настроек

```typescript jsx
import React from 'react';
import { BlockOptions } from 'react-txt-editor';

const Component = () => (
    <BlockOptions
        isShow={false}
    />
);

export default Component;
```
