# BlockPopup

Документация по API для contentConverter

## Import

```typescript
import {
    blocksToDraft,
    draftToBlocks,
} from 'react-text-editor';
```

## Example

**Переводит из формата редактора в формат draftjs**

```typescript
import { blocksToDraft } from 'react-txt-editor';

blocksToDraft([
    {
        type: 'MARKDOWN',
        data: {
            content: '# Heading',
        },
    },
]);
```

**Переводит из формата draftjs в формат редактора**

```typescript
import { draftToBlocks } from 'react-txt-editor';

draftToBlocks({ blocks: [], entityMap: {} });
```
